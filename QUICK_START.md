## 🛠 快速定制

### **修改品牌名称**
编辑 `index.html` 第8行：
```html
<title>你的品牌名 - 极简热量追踪工具</title>
```

### **调整会员价格**
编辑 `src/utils/memberPlans.ts`：
```typescript
price: {
  monthly: 9.9,  // 修改月费
  yearly: 89,     // 修改年费
}
```

### **添加更多食物**
编辑 `src/utils/foodDatabase.ts`：
```typescript
{ id: 'new_food', name: '新食物', calorie: 100, unit: '100g', category: '主食' }
```
